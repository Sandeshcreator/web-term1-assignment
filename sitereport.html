I have chosen W3School to learn to code.After pushing my index page code on GitHub, I was not active on Github because I was facing problems with the code and layouts. I also needed to delete the portfolio branch on GitHub because there were many conflicts and it was also affecting the main branch, so I created the porfoliobranch on GitHub. While creating this website, I found this course entertaining and helpful for our future. I found our tutor very helpful and also gave us knowledge about how people work in the real world.Talking about designing and layout, I have taken an idea from Dribbble.I saw many light-theme websites, so I tried a little dark-theme on my website.
